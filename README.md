# roomy

A Room entity generator.

Go to your room.

## Getting Started
Install with npm:
```bash
$ npm install
```
Link the command line tool (roomy):
```bash
$ npm link
```
Run `roomy` in the directory that contains your room.json file:
```bash
$ roomy
```
